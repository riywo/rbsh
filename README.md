Yet Another shell script

## Usage

    echo("test")
    #=> "test\n"
    
    echo("test").grep("test")
    #=> "test\n"
    
    echo("test").grep("no")
    #=> "\n"

## Installation

Add this line to your application's Gemfile:

    gem 'rbsh'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install rbsh
